# 8. Что такое область видимости (Scope)?

`Scope` - это места откуда мы имеем доступ к переменным и функции
`Глобальный` - переменый и функция объя-я для общего пространство
`Локалбный` - п/ф объя-й внутри функции доступный только этой фунции и всем вложенным элементами 
`Блочная` - находиться внутри фигурных {} скобках (блока) if...else, for()

# 9. Разница между let, const, var?

`var` - разное область видимости, всплывает, вызывается до его объявление
`let` - область видимости блочное, всплытие до объяв вызывает ошибку.
`const` - область видимости блочное, всплытие до объяв вызывает ошибку. Невозможно изменить данные

# 16. Разница между e.preventDefault() и e.stopPropagation()?

`e.preventDefault()` - откл поведение элемента по умолчанию
                     -- исполь на ссылке при клике отмен-е перехода
                     -- иполь на form  при клике отмен-е отправку формы
                     
`e.stopPropagation()` - откл распространение событие
                     -- откл всплытие
                     -- откл погружение

# 28. Разница между stopPropogation и stopimmediatePropogation?
     При наступление событие в баузее происхдит сплытие
`stopPropogation` - отменяет дальнейше всплытие, но на текущии элементе все оброботчики срабативает
`stopImmediatePropogation` - не только остоновит дальнейще всплытие но и остоновливает обоботку событие не текушим элементе

# 29. Разница между Load, DOMContentLoaded?
    Обе событие срабатывает при загузки веб страницы в браузера
`Load` -  это когда браузер загрузиль HTML и все зависимые ресурсы
`DOMContentLoaded` - отработает когда браузе поль-ю загрузиль HTML после чего построен DOM дерево, однако внешные ресурсы (стиль, скрипты, картинки) не загружены

# 30. Разница между Синхр / Асинхр function?

    Синхронная функция блокируещим а асинхр функция нет
`Синхр` - во время выполнение Js кода интерпретатор натыкается на синхр.функцию он блокиреут даль-е выполнение операций ожидание завершение синхр.функцию
`Асинхр`- не блокирует выполнение даль-е операций 
        ** запрос данных делает асинхронно
